# 🏺 Subrata Mondal: The 18-Node AI Ecosystem
**Craftsman | AI Web Developer | Sand Blaster**

This repository contains 18 high-end UI/UX components built with CSS 3D Transforms, Canvas API, and Google Apps Script.



## 🛠 Deployment Instructions
1. **Google Sheets:** Create a new sheet. Name three tabs: `Logs`, `Security_Logs`, and `Services`.
2. **Apps Script:** Go to `Extensions > Apps Script`. Paste the `Code.gs` provided.
3. **HTML Files:** Save each of your 18 UI components as separate files (e.g., `luxe.html`, `stark.html`).
4. **Deploy:** Click `Deploy > New Deployment`. Select `Web App`. Set access to `Anyone`.
5. **Connect:** Replace `YOUR_DEPLOYED_WEBAPP_URL` in the scripts with your live URL.

## 📡 Messaging Logic
Every interaction triggers a `doPost` event. When a user selects a service in the **Vault** or initializes the **Stark Core**, a signal is sent to the `Security_Logs` tab in Google Sheets for real-time monitoring.
